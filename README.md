A professional, responsive website for MBBS admission consultancy services in Uzbekistan. This website is designed to help medical aspirants learn about studying MBBS in Uzbekistan and connect with admission consultants.

## 🛠️ Setup Instructions

1. **Download/Clone the files** to your web server or local environment

2. **Replace the logo**: 
   - Add your company logo as `logo.png` (40x40px recommended)
   - Update the logo path in all HTML files if needed

3. **Customize content**:
   - Update contact information in all files
   - Replace placeholder phone numbers and email addresses
   - Add your actual office address
   - Update the Google Maps embed with your location

4. **Configure contact form**:
   - The contact form currently shows a success message
   - Connect it to your backend/email service
   - Update the form submission handling in `script.js`

5. **Update university information**:
   - Verify and update university details
   - Add more universities if needed
   - Update the featured university link to the correct website

## 🎨 Customization

### Colors
The website uses a medical/education color scheme defined in CSS variables:
- Primary: Blue (#2563eb)
- Secondary: Green (#059669)
- Accent: Red (#dc2626)

To change colors, update the CSS variables in `style.css`:

```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --accent-color: #your-color;
}
```
